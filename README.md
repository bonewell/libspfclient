# libspfclient
The library to write client application to use SPF microservice

## Build
```Shell
$ mkdir build
$ cd build
$ conan install ..
$ cmake ..
$ cmake --build .
```

## Run tests
```Shell
$ ./bin/spfclient_unittest
```

## Run command line sample
```Shell
$ ./bin/spfclient_cmd
```
